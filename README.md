# Campus Placement Prediction System

This project predicts whether a student will get placed based on academic and skill details.

## Technologies Used
- Python
- Flask
- Machine Learning
- Scikit-learn
- HTML

## Features
- Student placement prediction
- Web interface
- Machine learning model integration

## How to Run
1. Install requirements
2. Run python campus placement system.py
3. Open http://127.0.0.1:5000
